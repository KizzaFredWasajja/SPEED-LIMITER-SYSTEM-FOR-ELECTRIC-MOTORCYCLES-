# SPEED-LIMITER-SYSTEM-FOR-ELECTRIC-MOTORCYCLES-
Control speed of overspeeding electric motor bikes by maintaining motor speed in a safe range for specific busy areas.
# PROBLEM
Overspeeding motorbikes especially to those delivery riders that tend to speed in order to make more shifts a day, or other road riders that recklessly ride that put road user's lives at risk.
# SOLUTION
Controlling the speed of the motor maintaining it withing a safe riding range, keeping all road user's lives safe including the riders and also for safety of all vehicles.
# SETUP INSTRUCTIONS.
- System powered.
- Speed control of the motor using a potentiometer knob.
- monitor speed and warning message on the display.
- Realise motor won't exceed a preset limit ,of 0.16km/ Jr as set in the project
